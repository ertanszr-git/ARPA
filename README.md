# ARP Poisoning/Spoofing Alert Tool - ARPA

ARPA is a script for detecting Arp poisoning/spoofing and recording it.

## Installation

```bash
git clone https://github.com/ertanszr-git/ARPA
cd ARPA
chmod 500 install.sh
```

## Usage

```bash
arpa
```
